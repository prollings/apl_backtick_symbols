# README

All this does is provide Dyalog-style backtick-symbols.  
eg.

- ``` `` => ⋄```
- ``` `~ => ⌺```

Some input characters are mapped to the same outut symbol.  
eg.

- ``` `a or `A => ⍺```
- ``` `w or `W => ⍵```
